## Run Locally

Clone the repository:

```bash
git clone https://github.com/Jayani22/Portfolio.git
```

Navigate to the project folder:

```bash
cd Portfolio
```

Open `index.html` in your browser.

## Run with Docker

Build the Docker image:

```bash
docker build -t portfolio .
```

Run the container:

```bash
docker run -d -p 8080:80 portfolio
```

Open your browser and visit:

```
http://localhost:8080
```

## GitHub Actions

This repository includes a GitHub Actions workflow that automatically deploys the portfolio to GitHub Pages whenever changes are pushed to the main branch.
